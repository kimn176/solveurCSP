## Les commandes permettant de le compiler et de l’exécuter des ficher jar (à exécuter dans la racine du dossier)
   - Pour le solverCSP:
	java -jar distrib/solvercsp.jar
	
   - Pour l'application Sudoku:
   	java -jar distrib/sudoku.jar
   	
    - Pour l'application N-reines:
   	java -jar distrib/nreines.jar
